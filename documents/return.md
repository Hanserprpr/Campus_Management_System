# API 接口返回值

| Code | 描述                                               |
| ---- | -------------------------------------------------- |
| 200  | 请求成功                                           |
| 400 | 请求参数错误或失效 |
| 401 | 验证失败 |
| 403 | 禁止访问 |
| 404 | 访问内容不存在或已删除 |
| 409 | 与现有资源冲突（如邮箱已注册） |
| 500 | 服务器内部错误 |
| 554 | 邮件发送失败 |
